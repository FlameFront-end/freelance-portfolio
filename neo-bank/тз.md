# NeoBank — Personal Finance Manager (Concept UI)
## Fintech / Neobank Web App — Full Technical Specification

---

## 0. Цель проекта

Создать **UI-heavy интерфейс цифрового банка / персонального финансового менеджера**, который выглядит как реальный коммерческий fintech-продукт уровня $5–10k, но:

- полностью frontend-only
- без реальных банков / API
- без платежей
- без NDA-рисков

Проект должен демонстрировать:
- сложный, но чистый интерфейс
- работу с данными (балансы, операции, категории)
- графики и отчёты
- состояние денег в реальном времени (визуально)

Рабочее название: **NeoBank**
Подзаголовок: **Personal Finance Manager (Concept UI)**

---

## 1. Стек
