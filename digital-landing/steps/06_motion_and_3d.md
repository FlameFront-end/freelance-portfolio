# Шаг 6. Анимации и 3D

## Цель
Добавить обязательную motion-часть без потери плавности.

## Задачи
- [ ] Hero: fade/slide-in для текста, CTA и визуального блока.
- [ ] Features: stagger-анимация при входе в viewport.
- [ ] Showcase tabs: плавная смена контента.
- [ ] FAQ: анимация открытия/закрытия по высоте.
- [ ] Navbar: sticky blur + изменение opacity при скролле.
- [ ] 3D-решение:
  - [ ] Вариант A: Spline embed + lazy load + фолбэк.
  - [ ] или Вариант B: облегченный Three.js.

## Критерий готовности
- Длительности в коридоре 0.35-0.6s.
- Нет заметных лагов на типичном ноутбуке и мобильном.
