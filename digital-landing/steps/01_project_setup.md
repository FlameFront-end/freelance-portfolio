# Шаг 1. Project Setup

## Цель
Поднять базовый проект на требуемом стеке и подготовить структуру.

## Задачи
- [ ] Инициализировать Next.js (App Router) + TypeScript.
- [ ] Подключить TailwindCSS.
- [ ] Установить `framer-motion`.
- [ ] Подготовить структуру:
  - [ ] `app/`
  - [ ] `components/`
  - [ ] `content/`
  - [ ] `lib/`
  - [ ] `styles/`
- [ ] Настроить `next/font` для основной типографики.
- [ ] Проверить старт проекта (`dev` и `build` без ошибок).

## Критерий готовности
- Проект запускается локально.
- Основные зависимости подключены.
- Папки под дальнейшую разработку созданы.
