# Шаг 7. Адаптив и производительность

## Цель
Довести UI до production-ready поведения на разных экранах.

## Задачи
- [ ] Проверить брейкпоинты:
  - [ ] mobile `375px`
  - [ ] tablet `768px`
  - [ ] desktop `>=1280px`
- [ ] На mobile:
  - [ ] 3D скрывается или заменяется на легкий блоб
  - [ ] навигация переходит в burger
- [ ] Использовать `next/image` где есть изображения.
- [ ] Убедиться, что тяжелые скрипты грузятся лениво.
- [ ] Прогнать Lighthouse и достичь `Performance >= 90` на `/`.

## Критерий готовности
- Верстка не ломается на ключевых разрешениях.
- Lighthouse по производительности соответствует ТЗ.
