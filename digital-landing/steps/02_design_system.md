# Шаг 2. Design System

## Цель
Собрать базовый визуальный каркас в стиле premium tech / glass.

## Задачи
- [ ] Определить токены в `globals.css`:
  - [ ] цвета фона/текста
  - [ ] градиенты
  - [ ] glow-цвета
  - [ ] border/тени
- [ ] Сделать reusable-классы для glass-блоков.
- [ ] Настроить типографическую шкалу:
  - [ ] display заголовок
  - [ ] H2/H3
  - [ ] body/small
- [ ] Сделать базовые UI-компоненты:
  - [ ] `Button` (primary/secondary)
  - [ ] `Section` wrapper
  - [ ] `Card` (glass)

## Критерий готовности
- На странице есть единый стиль.
- Новые секции можно собирать из готовых примитивов.
