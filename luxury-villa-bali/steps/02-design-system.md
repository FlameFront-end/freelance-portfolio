# Step 02 - Design System

## Goal
Set premium visual language before building sections.

## Tasks
- Choose one theme variant from spec:
  - Luxury Light (recommended), or
  - Luxury Dark
- Add CSS variables in `src/styles/theme.css`:
  - `--bg`, `--panel`, `--text`, `--text-2`, `--muted`, `--border`, `--accent`, `--accent-soft`
- Configure two font families:
  - Display: Playfair Display or similar
  - Body: Inter/Manrope/DM Sans
- Define typography scale:
  - H1 desktop/mobile ranges
  - H2 range
  - body and meta
- Add global spacing system and container widths.

## Done When
- Theme, fonts, and typography are applied globally.
- Base UI already looks premium before section content.
