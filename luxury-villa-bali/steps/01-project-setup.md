# Step 01 - Project Setup

## Goal
Create a clean Vite + React + TypeScript base and folder structure.

## Tasks
- Initialize app with Vite (`react-ts`).
- Install required deps:
  - `framer-motion`
  - `sass`
  - `yet-another-react-lightbox`
- Create base folders:
  - `src/components/layout`
  - `src/components/sections`
  - `src/components/ui`
  - `src/components/gallery`
  - `src/components/booking`
  - `src/styles`
  - `src/data`
- Add route scaffold for:
  - `/`
  - `/villa`
  - `/book`

## Done When
- App runs with `npm run dev`.
- Route placeholders exist for all required pages.
