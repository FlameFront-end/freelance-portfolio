# Step 09 - Performance and Assets

## Goal
Keep the project fast and portfolio-ready.

## Tasks
- Use optimized image sizes and formats.
- Lazy-load gallery and lightbox where possible.
- Avoid heavy dependencies and unnecessary runtime work.
- Check Lighthouse and target performance score >= 90.
- Verify no layout shifts from image loading.

## Done When
- App remains smooth on mid-range mobile.
- Lighthouse performance is close to or above target.
