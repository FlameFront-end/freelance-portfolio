# Step 04 - Story and Highlights

## Goal
Implement storytelling blocks after the hero.

## Tasks
- Build parallax story section with:
  - villa background image
  - short copy lines:
    - `Designed for privacy`
    - `Sunset terrace`
    - `Natural materials`
- Implement parallax using:
  - CSS fixed background if stable, or
  - lightweight JS `translateY` from scroll position
- Add highlights section with 3 cards:
  - Private pool
  - Tropical garden
  - Sunset terrace
- Add minimal hover effect:
  - subtle darken/lighten
  - no aggressive scale animation

## Done When
- Parallax effect feels smooth and not heavy.
- 3 highlight cards are responsive and visually balanced.
