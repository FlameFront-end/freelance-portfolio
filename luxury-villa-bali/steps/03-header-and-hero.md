# Step 03 - Header and Hero

## Goal
Build the top of the landing page with strong first impression.

## Tasks
- Create sticky transparent header:
  - Logo: `AURA VILLA`
  - Nav: Villa, Gallery, Amenities, Location, Reviews, Book
  - CTA: `Check availability`
- Implement header state change on scroll:
  - solid background + blur/solid panel style
- Build hero section:
  - min height `90vh`
  - large background image + readable overlay gradient
  - H1 + subtitle + two CTAs
  - quick facts row (`3BR`, `6 guests`, `private pool`, `Pererenan`)
- Add subtle entrance animation (fade + small slide).

## Done When
- Header is usable and changes style after scroll.
- Hero content is readable and visually premium on desktop/mobile.
