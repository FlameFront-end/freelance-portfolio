# Step 05 - Gallery and Lightbox

## Goal
Create a polished gallery with mandatory lightbox behavior.

## Tasks
- Prepare gallery data in `src/data/gallery.ts` with 10+ images.
- Build responsive grid:
  - desktop: 2-3 columns
  - mobile: 1 column
- Add skeleton loading state (600-900ms simulated delay).
- Integrate `yet-another-react-lightbox` with:
  - next/prev controls
  - counter
  - keyboard navigation
- Ensure smooth open/close transitions.

## Done When
- Gallery loads with placeholders and opens in lightbox on click.
- Keyboard and counter behavior work correctly.
